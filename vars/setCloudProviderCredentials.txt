<p>
Set up credentials to access the cloud provider
</p>
<em>Parameters</em>
<p>
<dl>
<dt>credentialsId</dt>
<dd>
The id of the Jenkins credential to use
</dd>

<dd>
The name of the variable holding the name of the automation user
</dd>
</dl>
</p>
<em>Notes</em>
<p>
