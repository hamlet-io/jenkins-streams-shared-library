<p>
Send stream trigger failure notification to one or more collaboration channels
</p>
<em>Parameters</em>
<p>
<dl>
<dt>stream</dt>
<dd>
Stream that failed to trigger
</dd>
<dt>segment</dt>
<dd>
Segment
</dd>
<dt>group</dt>
<dd>
Deployment group
</dd>
<dt>deploymentUnits</dt>
<dd>
Deployment units
</dd>
<dt>codeUnits</dt>
<dd>
Code units
</dd>
<dt>commit</dt>
<dd>
Commit reference
</dd>
<dt>tag</dt>
<dd>
Commit tag
</dd>
<dt>channels</dt>
<dd>
Channels to be notified - see notifyChannels for channel formats
</dd>
</dl>
</p>
<em>Notes</em>
<p>
<ul>
<li>
this step is designed to be used in build jobs
</li>
</ul>
